# Frontend E-commerce Website 🛒

A **fully functional frontend e-commerce website** built with **HTML, CSS, and JavaScript**, featuring a modern UI, responsive layout, and interactive components.

This project simulates a real-world shopping experience, including featured products, product grids, cart management, wishlist, and checkout modal.

---

## 🔹 Features

- **Featured Products** – Highlighted top picks dynamically loaded from API  
- **Product Grid** – Browse all products with images, titles, and prices  
- **Add to Cart** – Manage product quantities in the cart  
- **Wishlist** – Toggle favorite products easily  
- **Checkout Modal** – Review cart items, update quantities, remove items, and see total price  
- **Responsive Design** – Optimized for desktop, tablet, and mobile screens  
- **Smooth Notifications** – Popup notifications for actions like adding to cart  

---

## 💻 Tech Stack

- **HTML5** – Semantic markup  
- **CSS3** – Flexbox, Grid, and modern styling  
- **JavaScript (ES6+)** – DOM manipulation, event handling, and API integration  
- **FakeStore API** – Dynamic product data  

---

## 🚀 How to Use

1. **Clone the repository**  
git clone [Repository](https://github.com/shahbazal0m/frontend-ecommerce-website)
2. Open index.html in your browser
3. Explore the app:
- Browse products
- Add items to cart
- Toggle wishlist
- Open checkout modal to see your cart
